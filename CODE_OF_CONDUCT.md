# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In short: be respectful, be constructive, and make this a welcoming community for everyone.

## Reporting

If you experience or witness unacceptable behavior, please use [GitHub's private reporting](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) or open a private security advisory on this repository.
